Planning

I have ensured accessibility for my website through 3 ways: Keyboard navigation: I have made sure that my website can be navigated using only the keyboard, without a mouse. This makes it more accessible for users with mobility impairments. I have also added descriptive sub-titles to all pages on the website, which helps users to quickly understand the content of each page. Color contrast: I have ensured that the color contrast between text and background is at least 4.5:1 for all text on my website. This makes it easier for users with visual impairments to read the text.

To make the website easy to use or navigate, I have created a clear and consistent navigation menu, which makes it easy for users to find the information they need. The menu is located in the same place on every page and it's easy to identify it by its design and color. Additionally, I have made sure that all the links on the website are clear and self-explanatory, so users can easily understand where they are going to be directed to.
